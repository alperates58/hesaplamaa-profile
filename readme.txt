Hesaplamaa kullanıcıları için kişisel analiz, profil dashboard ve paylaşılabilir profil kartları eklentisi.

== Description ==

Bu eklenti hesaplamaa.com için bağımsız bir profil dashboard sistemi sunar.

Özellikler:
* Premium görünümlü kişisel analiz dashboard'u
* Paylaşılabilir profil kartları
* Modül tabanlı analiz sistemi
* Astroloji, numeroloji, sağlık modülleri
* GitHub üzerinden otomatik güncelleme desteği
* Gizlilik odaklı tasarım

== Installation ==

1. `hesaplamaa-profile.zip` dosyasını WordPress Eklentiler > Yeni Ekle > ZIP Yükle ile kurun.
2. Eklentiyi etkinleştirin.
3. WordPress Admin > Hesaplamaa Profil menüsünden ayarları yapılandırın.
4. Profil dashboard shortcode'unu sayfanıza ekleyin: `[hap_user_profile_dashboard]`
